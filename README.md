# Hotel Booking Insights and Cancellation Prediction Using Data Analytics

## Description
This project analyzes hotel booking data to identify booking patterns, pricing trends, customer behavior, and factors influencing cancellations. It also includes a machine learning model to predict booking cancellations using historical data.

## Domain
Data Science / Data Analytics

## Dataset
Hotel booking dataset containing information such as lead time, arrival dates, stay duration, guest details, pricing (ADR), special requests, and cancellation status.

## Key Objectives
- Analyze booking trends and customer behavior  
- Study pricing variations across seasons  
- Understand factors affecting booking cancellations  
- Predict booking cancellations using machine learning  

## Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn, Plotly  
- Scikit-learn  

## Machine Learning Model
- Random Forest Classifier  
- Achieved accuracy of approximately **81%**
